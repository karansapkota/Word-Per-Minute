# Speed Typing Test
This Python code implements a simple speed typing test using the curses library. It presents a randomly selected text phrase from a file and measures your typing speed in words per minute (WPM).

**Features**
1. Displays target text and tracks your progress visually, highlighting correct and incorrect characters.
2. Calculates and displays your WPM in real-time.
3. Uses color-coding to indicate correct and incorrect characters.
4. Allows for multiple tests in a single session.
   
**Requirements**
* Python 3.x
* curses library (usually included in Python installations)
